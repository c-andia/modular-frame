A brief description of each part:

Innie + Outie: The main parts that make up a length of the frame. Alternating these interlocking parts allows you to adjust the size.
Extended Innie + Outie: The brackets extended by 30mm. 
L-joint: A 90 degree connector.
I-joint: A 90 degree connector that meshes orthogonally with other joints to create a 3-dimensional joint.
K- and C-joints: Angular joints.

Outie parts have more than two holes. The inner holes with the hex couterbores allow a hex nut to be pushed through and external parts to be attached using an M6 bolt. The outer ones are for attaching innie parts.

#### Note: parts followed by 'l' and 'r' have no symmetry. This means that you must verify which direction of part is required or you will have considerable difficulty when it comes time to assemble.
